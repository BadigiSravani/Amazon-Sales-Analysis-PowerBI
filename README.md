# 📊 Amazon Sales Analysis Dashboard | Power BI

## 📌 Project Overview
This project is an interactive Power BI dashboard built using an Amazon sales dataset from Kaggle.  
It helps analyze product performance, pricing trends, discounts, ratings, and customer behavior.

The main goal is to convert raw sales data into meaningful business insights using data visualization.

---

## 🎯 Objective
- Analyze Amazon product sales data  
- Identify top-rated products  
- Analyze customer reviews and ratings  
- Build an interactive dashboard for insights  

---

## 🛠️ Tools & Technologies Used
- 📊 Power BI Desktop  
- 🧹 Power Query (Data Cleaning & Transformation)  
- 📈 DAX (Data Modeling & Calculations)  
- 📂 Kaggle Dataset  

---

## 📊 Key Insights
- Top performing product categories  
- Average discount percentage across products  
- Relationship between price, discount, and rating  
- Most reviewed products  
- Customer rating distribution  

---

## 📌 Dataset Source
The dataset used in this project is from Kaggle:

👉 https://www.kaggle.com/datasets/karkavelrajaj/amazon-sales-dataset

This dataset contains product details such as price, discount, rating, and review data used for analysis.
